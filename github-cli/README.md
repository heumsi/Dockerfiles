# Github CLI

This is a custom [github cli](https://github.com/cli/cli#installation) dockerfile.

## Usage

```bash
$ docker run heumsi/github-cli:2.13.0 gh --version

gh version 2.13.0 (2022-06-22)
https://github.com/cli/cli/releases/tag/v2.13.0
```
